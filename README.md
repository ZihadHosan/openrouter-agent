# VS Code AI Extension

Workspace for **OpenRouter Agent** — a lightweight VS Code/Cursor extension with Ask, Plan, and Agent chat modes powered by the [OpenRouter](https://openrouter.ai/) API.

## Project

See [`openrouter-agent/README.md`](openrouter-agent/README.md) for setup, development, and usage.

```bash
cd openrouter-agent
npm install
npm run compile
```

Press **F5** in VS Code to launch the Extension Development Host.
